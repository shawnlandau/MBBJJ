## ğŸŒ Deployment to Netlify

### Option 1: Deploy via Git (Recommended)

1. Push your code to GitHub/GitLab/Bitbucket
2. Log in to [Netlify](https://app.netlify.com)
3. Click "Add new site" â†’ "Import an existing project"
4. Connect your Git provider and select the repository
5. Netlify will auto-detect the build settings from `netlify.toml`
6. Click "Deploy site"

### Option 2: Manual Deploy

```bash
# Build the site
npm run build

# Install Netlify CLI
npm install -g netlify-cli

# Deploy
netlify deploy --prod --dir=dist
```

### Custom Domain Setup

1. In Netlify dashboard, go to "Domain settings"
2. Click "Add custom domain"
3. Enter `myrtlebeachbjj.com`
4. Follow DNS configuration instructions
5. SSL certificate will be automatically provisioned

## ğŸ¨ Customization

### Update Site Information

Edit `src/utils/constants.ts` to update:
- Site name and contact information
- Address and hours
- Social media links
- Programs and schedules
- Testimonials
- FAQ items

### Change Colors

Edit `src/styles/global.css` CSS variables:
```css
:root {
  --color-primary: #1a1a1a;
  --color-secondary: #c41e3a;
  --color-accent: #f4f4f4;
  /* ... */
}
```

### Add Images

Place images in the appropriate `public/images/` subdirectories:
- `hero/` - Hero section backgrounds
- `programs/` - Program images
- `instructors/` - Instructor photos
- `gallery/` - General photos
